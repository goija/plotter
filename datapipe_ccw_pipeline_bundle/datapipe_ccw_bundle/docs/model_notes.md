# Model notes

## Canonical cycle
For each `(n,k)`, use the lexicographically least de Bruijn cycle generated by the Lyndon/FKM construction.

## Disturbance channels
Per state:
- `dist_weather`
- `dist_atmospheric`

Combined disturbance:
- `D^(c) = (1-c) * W + c * A`
- `c in {0.0, 0.1, ..., 1.0}`

## Continuous color weight
- `ccw = c`
- use `ccw` for image/graph interpolation only
- do not use red in this gradient because red is reserved for operational errors

## Report classification
- `c <= 0.2` -> `weather_dominates`, `blue`
- `0.3 <= c <= 0.7` -> `hybrid`, `purple`
- `c >= 0.8` -> `atmospheric_dominates`, `orange`
